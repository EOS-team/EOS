== ConfiguringEdition ==
Unity VCS은 두 가지 에디션으로 제공됩니다. 하나는 높은 확장성이 필요한 대규모 팀에 적합한 Enterprise 에디션이며, 다른 하나는 최대 15명으로 구성된 개발자 그룹을 위한 Team 에디션입니다. plasticscm.com에서 두 에디션에 대한 자세한 내용을 알아볼 수 있습니다

== ChooseEdition ==
설치할 에디션을 선택하십시오(숫자 입력)
